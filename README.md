# Vida com menos estresse

## 📒 Descrição
Este e-book apresenta um guia prático para viver uma vida mais leve e com menos estresse, abordando hábitos, organização pessoal, saúde mental e estratégias simples que podem ser aplicadas no dia a dia para melhorar a qualidade de vida.

## 🤖 Tecnologias Utilizadas
IA Generativa ChatGPT para criação e estruturação do conteúdo;
Ferramenta LibreOffice (ou similar) para edição e formatação do e-book.
## 🧐 Processo de Criação
Utilizei o ChatGPT para gerar a estrutura do e-book, organizar os capítulos e desenvolver conteúdos claros e objetivos sobre redução do estresse. Foram aplicadas técnicas de escrita simples e prática, com foco em fácil compreensão. A formatação final foi feita em um editor de texto, organizando títulos, seções e layout para melhor leitura.

## 🚀 Resultados
O e-book resultante é um material direto e útil, com dicas práticas que podem ser aplicadas imediatamente. Ele oferece valor tanto para quem busca melhorar a rotina quanto para quem deseja entender melhor como controlar o estresse.
